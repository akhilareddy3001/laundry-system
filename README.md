#Mini Laundary Order Management System
## Setup
1. Install Flask - pip install flask
2. Run - python app.py
## Features
-Create Order
-View Orders
-Update Status
-Dashboard

## APIs
POST /create_order
GET /orders
PUT /update_status/<id>
GET /dashboard
